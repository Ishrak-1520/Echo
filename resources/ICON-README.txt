Place your app icon files here:

  resources/
  ├── icon.ico     ← Windows app icon (256×256 recommended)
  ├── icon.png     ← PNG version (512×512 recommended)
  └── cursor.png   ← Optional: your custom cursor icon (36×36 to 64×64)

ICON TOOLS
──────────
• Convert PNG→ICO free: https://icoconvert.com
• Recommended icon size: 512×512 PNG, then convert to ICO
• Cursor icon: any square PNG/SVG, ideally 48×48 or smaller

CURSOR ICON
───────────
If you don't provide a cursor.png, Drishti uses its built-in
glowing-eye SVG (the দৃষ্টি symbol). You can also set a custom
cursor image any time from the Settings panel inside the app.

QUICK PLACEHOLDER
─────────────────
For development, the app will run without icons (tray will be blank).
Add proper icons before distributing / building the installer.
